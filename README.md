# Sprint0
Sprint 0: Conocimientos previos a especialidad PHP

La idea de este ejercio es la interación con GitHub, primeros pasos:

• Nivel 1:
  - Ejercicio 1. 
    Crea tu propia cuenta en Github. Una vez creada la cuenta, crea un repositorio. Decide tú mismo/a cómo querrás organizarte. Nosotros recomendamos, bien la creación de un repositorio por actividad entregable o bien un repositorio por Sprint.

  - Ejercicio 2
    Clona el repositorio en tu ordenador, en la carpeta donde vas a trabajar los diferentes ejercicios de la especialidad.

    Entrega tanto el link de tu repositorio Github como una captura de pantalla que muestre cómo lo has clonado en tu ordenador.

• Nivel 2
  - Ejercicio 1
    Crea archivos de prueba (con el contenido que quieras) para practicar con las versiones de GIT en tu ordenador. Versiona distintos cambios.

    Entrega una captura de pantalla.


• Nivel 3
  - Ejercicio 1
    Sube las versiones que has generado en tu repositorio Github. Con el link del ejercicio 1 del Nivel 1 quedará claro si lo has hecho correctamente.
